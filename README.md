# 🧠 AutoPes — Automação de Pesquisas com PyAutoGUI

Este projeto executa pesquisas automáticas no navegador Microsoft Edge com base em temas e perguntas pré-definidas.  
Ele foi projetado para simular pesquisas humanas, registrar resultados e oferecer uma interface gráfica simples e intuitiva.

---

## 🚀 Funcionalidades

- Abre automaticamente o navegador **Microsoft Edge**
- Gera perguntas aleatórias sobre diversos temas
- Realiza pesquisas automaticamente
- Fecha as abas após cada pesquisa
- Salva os resultados em um arquivo `.csv`
- Interface gráfica feita com **Tkinter**
- Logs detalhados em arquivo `.log`
- 100% offline — não precisa de Selenium nem WebDrivers

---

## 🧩 Tecnologias utilizadas

- **Python 3.10+**
- **PyAutoGUI** → automação de mouse e teclado  
- **Requests** → verificação de conectividade  
- **Tkinter** → interface gráfica  
- **CSV / Logging** → registro de resultados e logs
